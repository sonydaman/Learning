The Element The element specifies a list of pre-defined options for an element. Users will see a drop-down list of the pre-defined options as they input data. The list attribute of the element, must refer to the id attribute of the element.

The datalist Element

The datalist element specifies a list of pre-defined options for an input element.

Note: The datalist tag is not supported in Safari prior version 12.1.
